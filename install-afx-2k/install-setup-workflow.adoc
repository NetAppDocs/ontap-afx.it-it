= Flusso di lavoro di installazione e configurazione per i sistemi storage AFX 2K
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Per installare e configurare il tuo sistema storage AFX 2K, devi verificare i requisiti hardware, preparare il sito, installare gli switch, installare e cablare i componenti hardware, accendere il sistema e configurare il tuo cluster ONTAP AFX.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Uno"]link:../install-setup/install-setup-requirements.html["Esaminare i requisiti di installazione hardware"]
[role="quick-margin-para"]
Verifica i requisiti hardware per installare il tuo sistema storage AFX 2K.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Due"]link:../install-setup/prepare-hardware.html["Preparati a installare il tuo sistema storage AFX 2K"]
[role="quick-margin-para"]
Preparati a installare il tuo sistema storage AFX 2K predisponendo il sito, verificando i requisiti ambientali ed elettrici, assicurandoti che ci sia spazio rack sufficiente, disimballando l'attrezzatura, verificando il contenuto con la bolla di accompagnamento e registrando l'hardware per il supporto.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Tre"]link:install-switches.html["Installa gli switch per il tuo sistema storage AFX 2K"]
[role="quick-margin-para"]
Installa gli switch Cisco Nexus 9808, 9332D-GX2B o 9364D-GX2A nell'armadio o nel rack. Installa un kit di pannello passante se usi lo switch Cisco Nexus 9364D-GX2A.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Quattro"]link:deploy-hardware.html["Installa l'hardware per il tuo sistema storage AFX 2K"]
[role="quick-margin-para"]
Installa i kit di binari per il tuo sistema di stoccaggio e i tuoi scaffali.  Metti in sicurezza il tuo sistema di archiviazione nell'armadio o nel rack per telecomunicazioni.  Successivamente, far scorrere i ripiani sulle guide installate.  Infine, fissare i dispositivi di gestione dei cavi sul retro del sistema di archiviazione per organizzare il passaggio dei cavi.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Cinque"]link:cable-hardware.html["Collega i controller e i ripiani del tuo sistema storage AFX 2K"]
[role="quick-margin-para"]
Per cablare l'hardware, collegare prima i nodi del controller di storage alla rete, quindi collegare i nodi del controller e gli scaffali di storage agli switch del cluster.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["Sei"]link:power-on-configure-switch.html["Accendi e configura gli switch per il tuo sistema storage AFX 2K"]
[role="quick-margin-para"]
Accendi e configura gli switch per il tuo sistema storage AFX 2K. Controlla le istruzioni di configurazione per gli switch Cisco Nexus 9808, 9332D-GX2B e 9364D-GX2A.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-7.png["Sette"]link:power-on-hardware.html["Accendi il tuo sistema storage AFX 2K"]
[role="quick-margin-para"]
Accendere ogni ripiano di archiviazione e assegnare un ID ripiano univoco prima di accendere i nodi del controller per identificare chiaramente ogni ripiano nella configurazione.
